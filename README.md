# FloxHub - Sistema de Fluxo Inteligente de Atendimento

## Sobre o Projeto

FloxHub é um CRM e Gestor de Atendimento Omnicanal para Governo, desenvolvido para otimizar o atendimento ao cidadão através de fluxos inteligentes e automação.

## Funcionalidades

- **Dashboard**: Visão geral das métricas de atendimento
- **Inbox**: Central de atendimento com suporte a WhatsApp
- **Contatos**: Gestão completa de cidadãos
- **Campanhas**: Disparos em massa de mensagens
- **Templates**: Modelos de mensagens padronizadas
- **Automações**: Regras automáticas de atendimento
- **Tickets**: Sistema de chamados e demandas
- **IA**: Assistente inteligente para atendimento

## Tecnologias

- React + TypeScript
- Tailwind CSS
- Supabase (Backend)
- WhatsApp Business API

## Como Executar

```bash
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev
```
